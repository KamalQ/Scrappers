# Scrapper
Various scrapers for different sites

- Scrapes events from https://coinmarketcal.com
- Scrapes Market Prices from https://ca.investing.com/crypto/ 

Requirements
- Chrome Web Driver
- Selenium
- Pandas
- Pymongo
- MongoDB
